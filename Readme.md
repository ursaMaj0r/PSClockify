# PSClockify

PowerShell module wrapper for Clockify API


## Installation

```
# Import the module
Import-Module "path_to_module"

# Connect to Instance
Connect-Session -apiKey "clockify_api_key" -Workspace "Workspace Name"

# Get Module Commands
Get-Command -Module PSClockify
```

### Prerequisites
- PowerShell Core
- valid Clockify API Key
